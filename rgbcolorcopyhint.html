使い方

このツールは「色を探す・コピーする」「数値から色を確認する」ことができるシンプルなカラーツールです。

色 → 数値（カラーコードをコピー）

画面に表示されている色一覧から好きな色を選びます。

  • シングルタップ：カラーコードをコピー
  • ダブルタップ：色を全画面でプレビュー

コピー形式は上のボタンで切り替えできます。

  • R,G,B → 255,0,0
  • HEX → #ff0000

数値 → 色（カラーコードから色を表示）

上のスイッチをONにすると入力モードになります。

入力欄にカラーコードを入力して「追加」を押してください。

対応形式：

  • HEX → #ff0000 / #f00
  • RGB → 255,0,0

入力した色は履歴に追加され、タップでコピーやプレビューができます。

履歴機能

  • 最大50件まで保存
  • 新しいものが上に表示
  • 同じ色は重複しない
  • ブラウザのデータ削除で消えます

コツ

  • ダブルタップで全画面表示すると色確認しやすい
  • よく使う色は履歴に残すと便利
